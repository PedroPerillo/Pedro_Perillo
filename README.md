# Pedro_Perillo

Project scaffold for lab work. Stores CSV inputs in `data/` and logs in
`logs/`.

## Repository layout
- `data/`: two CSV files provided for the lab
- `logs/`: output logs from the notebook
- `config.yaml`: optional configuration settings
- `RUN.md`: exact reproduction steps

## How to run
1. Place the two CSV files in `data/`.
2. Run the notebook as instructed in `RUN.md`.
